# Birthday Website Assets

Change the birthday girl's name in `script.js`:

```js
birthdayName: "Her Name"
```

## Videos

Put videos inside `assets/videos/` and update `assets/people.json`.

Recommended video format:

- `.mp4`
- portrait or landscape both work
- 15 to 60 seconds each
- keep each file under 25 MB if you plan to host it online

Example:

```json
{
  "name": "Aarav",
  "relation": "school best friend",
  "video": "assets/videos/aarav.mp4",
  "note": "A birthday message from Aarav.",
  "color": "#ffd4bd"
}
```

## Photos

This first version does not require photos. If you want the next version to feel more personal, add:

- 1 cute main photo of her for the hero
- 4 to 8 candid/funny photos for a memory gallery
- optional childhood photo for a surprise reveal section

Use simple filenames like `hero.jpg`, `memory-1.jpg`, `memory-2.jpg`.
